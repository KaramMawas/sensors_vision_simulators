# Contributing

Thank you for your interest in contributing to this repository.

## How to contribute

- Fork the repository
- Create a feature branch
- Make your changes
- Add or update tests when appropriate
- Submit a pull request

## Coding guidelines

- Write clear, modular Python code
- Document public functions and classes
- Prefer readable engineering-oriented naming
- Keep educational explanations where useful
- Preserve compatibility with the existing repository structure

## Reporting issues

Please use GitHub Issues to report bugs, request features, or ask technical questions.
