# Depth Camera

This module contains simulators for depth cameras and LiDAR-based time-of-flight systems.

## Topics

- LiDAR hardware architecture
- time-of-flight computation
- depth map formation
- point cloud reconstruction
- reflectivity and intensity maps
- animated live scanning

## Main scripts

- `src/lidar_depth_camera_live_sim.py`

## Run

```bash
python src/lidar_depth_camera_live_sim.py
```

## Structure

- `src/` - source code
- `notebooks/` - notebooks and demos
- `figures/` - images and plots
- `videos/` - GIFs or videos
- `data/` - example datasets
