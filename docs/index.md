## Purpose

This project provides educational and engineering simulators for:

- Depth sensing
- Time-of-flight and phase-shift measurement
- Photogrammetry pipelines
- Point cloud generation
- Spatial search
- 3D vision workflows
- Photography and Lenses distortion
