# Sensor Vision Simulators

Welcome to the project website for the **3D Vision Simulators** repository.

## Modules

- Camera Parameters
- Depth Camera
- KD-Tree
- Photogrammetry
- Structured Light Scanning
- Terrestrial Laser Scanning

## Purpose

This project provides educational and engineering simulators for:

- Depth sensing
- Time-of-flight and phase-shift measurement
- Photogrammetry pipelines
- Point cloud generation
- Spatial search
- 3D vision workflows
- Photography and Lenses distortion
